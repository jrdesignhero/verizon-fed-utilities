Marquee Loader - Database Lookup

To use the database lookup up to 9 content key ids and click load content keys. Each key content found in the database will load inside of the hero image container. Use the navigation menu on the right to close the view but keep the Marquee Loader running. Please remember to use a device identifier (-DESKTOP or -MOBILE) when requesting CKs for the loader tool. Example: CK_HP_MARQ_UPGRADE_GOOGLE_PIXEL_XL_TRADE_IN_05152017-DESKTOP.

Marquee Loader - HTML Upload

Here you can upload the HTML content directly to the hero container. Paste html in the container inside of [marquee][/marquee] tags. For example, [marquee]YOUR HTML CONTENT HERE[/marquee]. Click Insert Keys and your content will be uploaded directly to the page. Please Remember to only use HTML from the marquee maker tool at https://testman.verizonwireless.com/featured/digital-styleguide/autobot/marquee-maker-d/ or https://testman.verizonwireless.com/featured/digital-styleguide/autobot/marquee-maker-m/ or the tool tip will not work properly for the marquee. However, you can copy and paste the marquee code from the source of the homepage and see the marquee render properly, but the tool tips will not work on those marquees.
